# Startup Feedback Analyzer

A simple ML-powered application to analyze startup user feedback and classify sentiment.

## Why this project
This project was built to move beyond notebooks and understand what it takes to ship an ML model:
- data preprocessing
- model training
- dependency management
- deployment challenges

## Tech stack
- Python
- scikit-learn
- pandas
- Streamlit

## Status
Deployment in progress. Iterating and improving.
